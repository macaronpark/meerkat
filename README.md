# meerkat

## Installation

```bash
git clone https://github.com/macaronpark/meerkat.git
cd server
pnpm install
cd ../client
pnpm install
```

## Running the app

```bash
# meerkat/server
pnpm start

# meerkat/client
pnpm start
```

## Test

```bash
# meerkat/server
pnpm test

# meerkat/client
pnpm test
```
